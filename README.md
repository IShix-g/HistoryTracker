# HistoryTracker
history management and state restoration plugin for Unity. It records changes to data and states, allowing you to easily revert to any previous state at any time.

## Getting Started

### Install from Git URL

"Unity Editor : Window > Package Manager > Add package from git URL...".

```
https://github.com/IShix-g/HistoryTracker.git?path=Packages/HistoryTracker
```
